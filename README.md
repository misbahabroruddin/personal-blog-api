# Personal Blog API
an API for a simple personal blog application

## Getting Started
* first 
```bash
npm install
```
* Create .env file
* Create database
 ```bash
npx sequelize-cli db:create
```
* Running Migration
```bash
npx sequelize-cli db:migrate
```
* Running the server
 ```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```
